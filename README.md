# bbsMusicShare
暮光方块论坛-音乐分享助手

显示QQ音乐/网易云音乐的ID 便于在暮光方块论坛上分享

https://bbs.tsfk.top


----------------------------------------------------

#### QQ音乐若不显示复制代码，请F5刷新。

#### 网易云由于页面布局，是直接点击复制的，不显示复制了什么代码。
